Step 7: Implementation Order
Start with S-Box - Just copy the 256-byte array
Implement AddRoundKey - Simplest (just XOR)
Implement SubBytes - Use S-Box lookup
Implement ShiftRows - Array manipulation
Implement MixColumns - Most complex (GF(2^8) math)
Implement Key Expansion - Uses RotWord, SubWord, Rcon
Put it all together - Main encryption loop

Key Points for Your Implementation:
S-Box is fixed - Don't generate it, just use the standard one
Start with a hardcoded key - Generate random keys later
Test each operation separately - Use known test vectors
MixColumns is the hardest part - It uses Galois Field arithmetic
State is a 4×4 matrix - But stored as 16-byte array column-wise
